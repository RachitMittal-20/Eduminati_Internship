## ▶️ How to run

### Prerequisites
- Node.js 18+
- npm

### Steps
1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build the project: `npm run build`
5. Start the application: `npm start`
